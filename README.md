# Resonify

Resonify is your one stop destination to learn everything in MUSIC!

You can check out [Resonify](https://resonify-redux.vercel.app/) here

## Run this project locally

```
$ git clone https://github.com/AshutoshM29/Resonify-Redux.git
$ cd Resonify-Redux
$ npm install
$ npm start
```

## Features

- Home page
- Authentication
  * Login
  * Sign up
  * Log out
- Video listing page
- Playlist management
- Like/Dislike videos
- Watch later videos
- Watch history
- Search videos
- Single video page
